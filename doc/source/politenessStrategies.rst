politenessStrategies
====================

Computes Politeness Strategies features. See Section 4 of this `paper <http://www.cs.cornell.edu/~cristian/Conversations_gone_awry.html>`_.

Example usage: `extracting politeness features and markers <https://github.com/CornellNLP/Cornell-Conversational-Analysis-Toolkit/blob/master/examples/politeness-strategies/Politeness%20Marker%20and%20Summarize%20Demo.ipynb>`_, `understanding the (mis)use of politeness strategies in conversations gone awry on Wikipedia <https://github.com/CornellNLP/Cornell-Conversational-Analysis-Toolkit/blob/master/examples/conversations-gone-awry/Conversations_Gone_Awry_Prediction.ipynb>`_
